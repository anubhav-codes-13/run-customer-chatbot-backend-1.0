Customer Support Chatbot Backend Quick Start Guide

Backend Setup

Get the AI-powered support engine running
in three steps.

1
Step 1
Install Dependencies

Installs the core libraries required for the API, AI agents, and server execution.

$ pip install -r requirements.txt uvicorn fastapi
2
Step 2
Initialize Database & Knowledge Base

Prepares the SQLite database and generates AI-searchable FAISS indices for product knowledge.

$ python db_creation.py
$ python create_faiss_index.py
3
Step 3
Launch the Backend API

Starts the live FastAPI server on port 8000, ready for the frontend to connect.

$ python -m uvicorn src.server:app --port 8000
Health check → localhost:8000/health
